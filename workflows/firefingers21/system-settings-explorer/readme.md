## Usage

View and search for System Settings panes via the `sse` keyword.

![Entering workflow keyword](images/keyword.png)

Select a settings pane to open in System Settings. Scroll through results or type to filter.

![Viewing main settings](images/mainSettings.png)

* <kbd>↩</kbd> Open selected pane in System Settings.
* <kbd>⌘</kbd><kbd>↩</kbd> Open nested pane directly in System Settings.

Nested panes, such as `General` and `Accessibility`, contain additional settings browsable within Alfred. These settings can also be selectively included in the main search results from the Workflow's Configuration.

![Viewing General Settings](images/generalSettings.png)

* <kbd>↩</kbd> Open selected pane in System Settings.
* <kbd>⇧</kbd><kbd>↩</kbd> Return to main search results.

Settings are searchable using natural language and related keywords. For example, `and` matches `&`, and `Mission Control` matches its parent setting `Desktop & Dock`.

![Searching for settings panes](images/touchid.png)

![Searching for settings panes](images/missionControl.png)

Configure the Hotkey as a faster shortcut to view settings panes.
