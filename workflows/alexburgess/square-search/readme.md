## Usage

Search [Square](https://squareup.com/) items via the `sq` keyword.

![Screenshot of a square item search in Alfred](images/item.png)

Search customers with the `sqc` keyword.

![Screenshot of a square customer directory search in Alfred](images/customer.png)

Search dashboard pages with the `sqd` keyword.

![Screenshot of a square dashboard menu search in Alfred](images/dashboard.png)
