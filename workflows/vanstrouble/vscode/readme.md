## Usage

Open a new Visual Studio Code window in your current space via `code` keyword.

![Open new VScode window](images/code.png)

Browse your recently opened projects, workspaces, and files via `coder` keyword.

![Open recent project](images/coder.png)

Search for any file or folder on your Mac and open it directly in VS Code via `codef` keyword.

![Search any file or folder to open in VScode](images/codef.png)

Open any folder in VS Code by typing its path. If you don't provide a path, the current Finder window will be opened in VS Code automatically via `codep` keyword. This action is also available via Universal Actions.

![Open any path in VScode](images/codep.png)

Search the VS Code Marketplace directly via `codes` keyword.

![Search extensions in VScode marketplace](images/codes.png)

List all your installed extensions via `codex` keyword.

![Showing installed extensions](images/codex.png)

Configure the Hotkeys for faster triggering.
