## Usage

Get search suggestions from [IMDb](https://imdb.com/) via the `imdb` keyword.

![IMDb in-line results](images/imdb.png)

* <kbd>↩</kbd> Open the IMDb page for the selected title.
* <kbd>⌘</kbd><kbd>↩</kbd> Copy the IMDb title ID to clipboard.
* <kbd>⌘</kbd><kbd>⇧</kbd><kbd>↩</kbd> Paste the IMDb title ID into the frontmost app.
